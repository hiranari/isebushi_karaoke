# テスト用音声ファイル一覧

このディレクトリには、ピッチ検出機能をテストするための様々な音声ファイルが含まれています。
各ファイルは特定の条件や状況でのピッチ検出の精度を検証するために用意されています。

## 1. 単音テスト (single_tones/)
基本となる単音のテストファイルです。各音程が正確に検出できるかを確認します。

## 2. 楽器別テスト (instruments/)
異なる楽器による同じ音程のテストファイルです。楽器の音色による影響を検証します。

### ピアノ (piano/)
- `C2_piano.wav` - C2音 (65.41 Hz)
- `C3_piano.wav` - C3音 (130.81 Hz)
- `C4_piano.wav` - C4音 (261.63 Hz)

### チェロ (cello/)
- `C2_cello.wav` - C2音
- `C3_cello.wav` - C3音

### ベース (bass/)
- ベース楽器による音源ファイル

## 3. 条件別テスト (conditions/)

### 音量テスト (volume/)
同じC3音を異なる音量レベルで録音したファイルです。
- `C3_0dB.wav` - 基準音量
- `C3_-3dB.wav` - -3dB
- `C3_-10dB.wav` - -10dB
- `C3_-20dB.wav` - -20dB

### 音の長さテスト (duration/)
同じC3音を異なる長さで録音したファイルです。
- `C3_100ms.wav` - 100ミリ秒
- `C3_500ms.wav` - 500ミリ秒
- `C3_2000ms.wav` - 2000ミリ秒（2秒）

### ノイズテスト (noise/)
背景ノイズがある状態での音声ファイルです。

## 4. 動的な音の変化テスト (dynamic/)

### ビブラート (vibrato/)
ビブラートを含む音声ファイルです。ピッチの周期的な揺れの検出を検証します。

### グリッサンド (glissando/)
音程が連続的に変化する音声ファイルです。

## 5. 音階テスト (scales/)
様々な音階パターンのテストファイルです。
- `C2_major.wav` - C2からの長音階
- `C2_minor.wav` - C2からの短音階
- `C3_major.wav` - C3からの長音階
- `C3_minor.wav` - C3からの短音階
- `C3_chromatic.wav` - C3からの半音階（クロマティック）

## 使用方法

1. 基本的なピッチ検出精度の確認
   - まず `single_tones/` の単音ファイルで基本的な精度を確認
   - 次に `instruments/` の各楽器での検出精度を確認

2. 特殊な条件下での検証
   - 音量の影響: `conditions/volume/` のファイルを使用
   - 音の長さの影響: `conditions/duration/` のファイルを使用
   - ノイズの影響: `conditions/noise/` のファイルを使用

3. 動的な音の検出精度
   - ビブラート検出: `dynamic/vibrato/` のファイルで検証
   - 音程の連続的変化: `dynamic/glissando/` のファイルで検証

4. 音階での検証
   - `scales/` のファイルを使用して、連続した音程変化の検出精度を確認

## 注意事項

- 各ファイルは44.1kHz, 16bitで録音されています
- ファイル名は基本的に「音程_条件.wav」の形式で命名されています
- C2（65.41 Hz）、C3（130.81 Hz）、C4（261.63 Hz）が基準となる音程です
